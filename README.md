## Description

Plum Island is  an orchestrator for proactive surface-exposure discovery. (https://github.com/D4-project/Plum-Island)
It assigns scanning tasks to agents, collects their results, and stores the data with full history. The collected information is queryable.
This repository contains all Nmap NSE dedicated rules.
